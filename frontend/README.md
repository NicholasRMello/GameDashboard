# Frontend - Game Dashboard

Aplicacao React + TypeScript para o dashboard de jogos.

## Requisitos

- Node.js 20+

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Variavel de ambiente

- `VITE_API_URL`: URL base da API Java (padrao recomendado: `http://localhost:8080`).

## Scripts

- `npm run dev`: desenvolvimento
- `npm run build`: build de producao
- `npm run preview`: visualizar build local
